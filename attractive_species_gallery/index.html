Attractive species gallery
The following species are listed in order of 'attractiveness' according to the number of images for that species (in February 2012) in the Flickr Image Search API (when searching on "<scientific name>" - only the top ranking 100 species are shown for each taxonomic group). This page uses data from: the Joint Research Centre of the European Commission for the charisma data; the Flickr Image API to display an image for each species; data from the Global Biodiversity Information Facility for species names in other languages. The charisma data for all 30,000 species is available from here.
