Following instructions from here https://www.youtube.com/watch?v=rQvOAnNvcNQ
1. Create a new node project
```
cd firebase_demo
npm init -y
```
2. Install Firebase
```
npm i firebase
```
3. Added the code files
4. Install serve
```
npm i serve
```
5. Start it in the src/ folder
```
npx serve -s src/
```
